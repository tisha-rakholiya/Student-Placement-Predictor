# 🎓 Student Placement Predictor

Predict karo ki student placed hoga ya nahi!

##  Model Performance
- Algorithm: Random Forest
- Accuracy: 79.07%
- Dataset: 215 students, 12 features

##  Features
- Placement probability prediction
- Feature importance visualization
- Personalized improvement tips
- Score comparison chart

##  Tech Stack
Python | Scikit-learn | Streamlit | Pandas | Matplotlib

##  How to Run
pip install streamlit scikit-learn pandas matplotlib numpy
streamlit run spp.py
